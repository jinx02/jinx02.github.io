Typera购买安装

官网购买Typera

按照常理来说，我十分推荐在官网Typora官网上直接去购买license code。当然如果你不想看全是英文的界面，我推荐在淘宝上通过第三方官方独家代理数码荔枝进行购买。购买后将邮箱与序列号输入进Typera软件即可。

意料之外的错误

因为我自己犯迷糊，将我的e-mail address 输少了一位。很自然的，我的邮箱收不到邮件。起初我以为是因为我在国内，邮件被某些原因阻挡在了外部，于是我询问数码荔枝的客服寻求解决方法。我通过官方人工客服以及与hi@typora.io联系。在一天后找回了自己的序列号。成功激活Typera。此外我推荐使用国外邮箱接受相关邮件，如果发现问题立刻与人工客服进行联系。

插入图片的偏好设置

同4.1 按住Crtrl+‘,’打开偏好设置，我的设置为[复制图片到 ./${filename}.assets 文件夹]。需要写博客的同学们在偏好设置里面应该根据自己需要勾选相应的选项。

上传图片的偏好设置

当你在 Typora 中插入本地图片时，前面提到的插入图片的语法![alt](src)中的src部分在 Markdown 中是以本地路径的方式保存的，如果你要把你的 Markdown 文件分享给别人，或者发布到网上，那么要把图片一起分享或者发布，这样就比较麻烦。而如果src是 URL 的话，你在分享的时候只要分享一个 Markdown 文件就可以了，只要能联网，Markdown 的渲染引擎就能根据 URL 找到你引用的图片。

图片上传服务

  • iPic (macOS，免费增值)
  • uPic (macOS，开源)
  • PicGo.app (macOS / Windows / Linux, 只有简体中文版)
  • PicGo (命令行应用) (Windows / Linux)
  • 自定义 (macOS / Windows / Linux)

我选择的是PicGo.app因为适用于windows，并且相较于命令行程序减少的入手的难度。

图床

我采用的图床是GIthub的免费图床，缺点就是速度比较慢。具体的操作步骤可以参考我提供的参考网站。
